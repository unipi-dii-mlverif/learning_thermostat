# A learning thermostat with automatic swap


## Clone this repo

Clone this repo locally

```bash
$ git clone https://github.com/lausdahl/SEAMS2023Artefact-fmiSWAP.git
```

Change to the repo directory before going to the next step.

## Build the image and run it

```bash
$ make launch
```


After this step completes, you should see in the containing folder two files ```result.png``` and ```result.pdf```.
